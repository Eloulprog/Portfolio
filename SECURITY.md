# Security Notes

## Publication policy

Before publishing this portfolio publicly:

- review all media and documents for personal or sensitive information
- avoid committing oversized source media files when a lighter export or external hosting is more appropriate
- keep private credentials, tokens, and API keys out of the repository

## Current repository notes

- no application secrets are intentionally stored in this project
- the contact email is public by design
- the original large English project video is excluded from Git tracking
